■━…‥・　このフォントについて　・‥…━■

このフォントは活字風でありながら、普通の明朝体のように使うことが出来るように作成
したフリーフォントです。

他の方が配布していた、同様のコンセプトで作成されたフォントが配布終了したため、替
わりになれば……と制作を始めた結果、別物になってしまいました。
代替にはなりませんが、より使いやすいフォントになったと思います。

製作するにあたり「源雲明朝」「明朝體 むつき」「EB Garamond」というオープンソース
フォントをお借りしております。詳しくは下記をご参照ください。


■━…‥・　入っているもの　・‥…━■

・SatsukiGendaiMincho-M.ttf
　　フォントファイル本体。インストールすることで「さつき源代明朝」を利用すること
　　が出来ます。

・OFLicense.txt
　　SIL Open Font License 1.1。このライセンスの許す範囲でフォントを利用可能です。

・readme_説明書.txt
　　このファイル。まず最初にお読みください。

■━…‥・　ファイル形式について　・‥…━■

各 OS のヘルプや解説サイトを参照してください。Windows の場合はフォントファイルを
右クリックして「インストール」を選ぶとインストール可能です。

Windows で既に古いバージョンを入れている場合は一度サインアウト・サインインしてか
らフォントを削除、その後にフォントをインストールし、もう一度サインインしなおして
から使用したほうが安定するようです。

■━…‥・　更新履歴　・‥…━■

Ver 1.04 - 18 / 11 / 11
・威沙というアプリで縦組み PDF ファイルを出力すると仮名が空白になるのを回避
 ― 参照グリフの設定を正しく処理できないもよう。アプリ側かフォント側か不明ですが
　　Ver 1.02 のように相性問題が出やすいようなので参照グリフ機能自体を使わないよ
　　うに変更しました。

Ver 1.03 - 18 / 09 / 09
・縦組み時の拗音・撥音の位置がおかしくなっていたのを修正
・濁点、半濁点の合成位置を修正

Ver 1.02 - 18 / 09 / 05
・サブセットフォントメーカーというアプリで縦組み用サブセットフォントを作成すると
　濁音・半濁音付き仮名が空白グリフになる問題に対処
 ― サブセットフォントメーカーの仕様なのか不明ですが、参照グリフが参照する先のグ
　　リフも参照グリフだった場合に、さらにその参照先グリフを取得できないもよう。

Ver 1.01 - 18 / 09 / 04
・em dash(—)が縦組み時に縦組み用字形に変わらないのを修正
 ― JIS X 0208/0213 では全角ダーシに em dash(—)を使うように規定されているが、
　　慣用的に horizontal bar(―)が使われているため、見落としてました。

・wave dash(〰)を縦組み時にwavy line(⌇)に置き換えるよう追加
 ― 本来は別の記号なのですが、利便性のために源暎フォントでは縦組み時に自動的に置
　　き換えるようにしているので共通仕様としました。
